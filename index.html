Lab Overview

Welcome to the Grosh Lab. This page provides an overview of the numerical frameworks developed over the past several years. Each project, along with its associated code and documentation, is maintained in a dedicated GitHub repository, with links provided below. This site serves as a centralized reference for all lab projects, some publicly accessible and others available only to authorized collaborators. Additional information about ongoing research activities can be found on our homepage.


Public Projects

  1. HairBundleLab

    HairBundleLab is a specialized software designed to simulate mammalian cochlear hair bundle responses, offering a platform to explore and visualize their intricate processes. Users can define mechanical, electrical, and geometric parameters to match experimental needs, making it a versatile tool for studying these essential auditory mechanisms.

Internal Projects

  1. Genetic Algorithm for Data Fitting

    This sample code performs parameter optimization for the three-row stereocilia model of the mammalian hair bundle, enabling quantitative fitting to experimental data. The framework is modular and can be readily adapted for other nonlinear or linear first-order, multi-degree-of-freedom systems. The repository includes detailed instructions to help you customize and implement the model for your specific requirements.
